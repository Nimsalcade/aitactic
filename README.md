# Soccer Tactical Board

A mobile-friendly tactical board for soccer/football coaches and enthusiasts. This web application allows you to create and visualize team formations and tactics on a soccer pitch.

## Features

- Add players to the pitch with a single tap
- Drag and drop players to position them
- Automatic player numbering and position assignment
- Responsive design for mobile devices
- Clean, modern UI with Tailwind CSS
- Touch and mouse support

## Getting Started

1. Simply open `index.html` in a modern web browser
2. No installation or build step required
3. Works best on mobile devices or in mobile device emulation mode

## How to Use

- Tap the "Add Player" button or tap anywhere on the pitch to add a player
- Drag players to position them on the pitch
- Use the "Clear All" button to remove all players
- Players are automatically assigned numbers and positions

## Technologies Used
